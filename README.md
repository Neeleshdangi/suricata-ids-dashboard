# Suricata IDS Dashboard

A GUI-based Intrusion Detection System (IDS) dashboard built with Python and Suricata for real-time network monitoring and alert visualization.

## Features
- Real-time alert monitoring
- GUI dashboard
- Threat severity classification
- Suricata integration
- Security event visualization

## Technologies Used
- Python
- Tkinter
- Suricata IDS

## Prerequisites

- Kali Linux
- Python 3
- Suricata

## Install Suricata

bash
sudo apt update
sudo apt install suricata -y


## Start Suricata

bash
sudo suricata -i eth0


> Replace eth0 with your network interface.

## Run Dashboard

bash
python3 suricata_gui.py
